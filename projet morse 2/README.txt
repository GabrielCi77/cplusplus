Pour lancer le programme sur vs code :

-Aller sur le dossier contenant les fichiers depuis le terminal
-Taper make sur le terminal

L'éxécutable est créé --> Maintenant faire ./morse pour lancer le programme

Une invite de commande est ouverte, vous laissant le choix d'encoder et de décoder vos programme : le programme écrira tout ce qu'il y a à savoir.

ATTENTION :

!!!!!!!!!!!!!!!!!!Tout écrire en majuscule.!!!!!!!!!!!!!!!!!!!!
Quand un fichier vous est demandé, mettez le chemin en entier.
Le fichier wav encodé depuis votre écrit aura pour nom morse.wav et sera directement sur le repértoire.
